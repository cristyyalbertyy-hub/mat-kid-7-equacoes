Para o Tutor

Este metodo usa a analogia "passar pela porta" para ajudar o aluno a compreender a troca de sinal com confianca.

Este nao e o metodo formal mais usado nas escolas (metodo da balanca), mas funciona como apoio intuitivo para chegar a solucao sem decorar regras.

As questoes servem para verificar se o aluno percebeu o raciocinio. Nao sao para memorizacao, mas para consolidar seguranca.

E recomendavel ouvir o podcast em conjunto com o aluno para reforcar o entendimento.
